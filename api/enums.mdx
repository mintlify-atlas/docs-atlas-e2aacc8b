# Enumeration Types

Enumeration classes used throughout python-pptx for specifying various properties and settings.

## Action Enumerations

### PP_ACTION_TYPE

Specifies the type of a mouse action (click or hover).

**Alias:** `PP_ACTION`

**Import:**
```python
from pptx.enum.action import PP_ACTION
```

**Values:**

<ResponseField name="PP_ACTION.NONE" type="0">
  No action is performed.
</ResponseField>

<ResponseField name="PP_ACTION.NEXT_SLIDE" type="1">
  Moves to the next slide.
</ResponseField>

<ResponseField name="PP_ACTION.PREVIOUS_SLIDE" type="2">
  Moves to the previous slide.
</ResponseField>

<ResponseField name="PP_ACTION.FIRST_SLIDE" type="3">
  Returns to the first slide.
</ResponseField>

<ResponseField name="PP_ACTION.LAST_SLIDE" type="4">
  Moves to the last slide.
</ResponseField>

<ResponseField name="PP_ACTION.LAST_SLIDE_VIEWED" type="5">
  Moves to the last slide viewed.
</ResponseField>

<ResponseField name="PP_ACTION.END_SHOW" type="6">
  Slide show ends.
</ResponseField>

<ResponseField name="PP_ACTION.HYPERLINK" type="7">
  Hyperlink to a URL.
</ResponseField>

<ResponseField name="PP_ACTION.RUN_MACRO" type="8">
  Runs a macro.
</ResponseField>

<ResponseField name="PP_ACTION.RUN_PROGRAM" type="9">
  Runs a program.
</ResponseField>

<ResponseField name="PP_ACTION.NAMED_SLIDE_SHOW" type="10">
  Runs the slideshow.
</ResponseField>

<ResponseField name="PP_ACTION.OLE_VERB" type="11">
  OLE Verb action.
</ResponseField>

<ResponseField name="PP_ACTION.PLAY" type="12">
  Begins the slideshow.
</ResponseField>

<ResponseField name="PP_ACTION.NAMED_SLIDE" type="101">
  Moves to slide specified by slide number.
</ResponseField>

<ResponseField name="PP_ACTION.OPEN_FILE" type="102">
  Opens the specified file.
</ResponseField>

**Example:**
```python
from pptx.enum.action import PP_ACTION

assert shape.click_action.action == PP_ACTION.HYPERLINK
```

### MSO_AUTO_SHAPE_TYPE

Specifies a type of AutoShape.

**Alias:** `MSO_SHAPE`

**Import:**
```python
from pptx.enum.shapes import MSO_SHAPE
from pptx.util import Inches

slide.shapes.add_shape(
    MSO_SHAPE.ROUNDED_RECTANGLE, Inches(1), Inches(1), Inches(2), Inches(1)
)
```

**Common Values:**
- `MSO_SHAPE.RECTANGLE` - Rectangle
- `MSO_SHAPE.ROUNDED_RECTANGLE` - Rounded rectangle
- `MSO_SHAPE.OVAL` - Oval/Ellipse
- `MSO_SHAPE.DIAMOND` - Diamond
- `MSO_SHAPE.RIGHT_ARROW` - Block arrow pointing right
- `MSO_SHAPE.DOWN_ARROW` - Block arrow pointing down

## Text Enumerations

### MSO_AUTO_SIZE

Determines the type of automatic sizing for text frames.

**Import:**
```python
from pptx.enum.text import MSO_AUTO_SIZE
```

**Values:**

<ResponseField name="MSO_AUTO_SIZE.NONE" type="0">
  No automatic sizing. Text can extend beyond shape bounds.
</ResponseField>

<ResponseField name="MSO_AUTO_SIZE.SHAPE_TO_FIT_TEXT" type="1">
  Shape height (and possibly width) adjusts to fit text.
</ResponseField>

<ResponseField name="MSO_AUTO_SIZE.TEXT_TO_FIT_SHAPE" type="2">
  Font size is reduced as necessary to fit text within shape.
</ResponseField>

**Example:**
```python
shape.text_frame.auto_size = MSO_AUTO_SIZE.TEXT_TO_FIT_SHAPE
```

from pptx.enum.text import PP_ALIGN

```python
from pptx.enum.dml import MSO_FILL
```

**Values:**

<ResponseField name="MSO_FILL.SOLID" type="1">
  Shape is filled with a solid color
</ResponseField>

<ResponseField name="MSO_FILL.PATTERNED" type="2">
  Shape is filled with a pattern
</ResponseField>

<ResponseField name="MSO_FILL.GRADIENT" type="3">
  Shape is filled with a gradient
</ResponseField>

<ResponseField name="MSO_FILL.TEXTURED" type="4">
  Shape is filled with a texture

<ResponseField name="MSO_FILL.BACKGROUND" type="5">
